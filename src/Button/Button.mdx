## Theme config props

These configs applies to all Button elements on the application

    // Changes primary button font color
    primaryButtonFontColor:  @String - [Default: '#fff']

    // Changes secondary button border size
    secondaryButtonFontColor:  @String - [Default: '#fff']

    // Defines if Buttons have uppercase font
    buttonUppercaseFont:  @Boolean - [Default: true]

    // Defines default button border radius
    buttonBorderRadius: @String - [Default: 0]

    // Shows button shadow to flat button
    buttonShadow: @Boolean - [Default: false]

    // Defines shadow amount
    buttonShadowAmount: @Number - [Default: 0.2]

    // Defines how much to darken or lighten button when hovering
    buttonHoverAmount: @Number - [Default: -0.1]

    // Defines how much to darken or lighten button when clicked
    buttonActiveAmount: @Number - [Default: 0.4]

    // Defines button hover style
    buttonFlatHovercolor: @Enum ['inverted'] - [Default: null]

    // Defines button animation transition
    buttonTransition: @Number - [Default: 0.2]
