## Theme config props

These configs applies to all Input elements on the application

    // Changes default input padding (unit: rem)
    inputPadding:  @Number - [Default: 1]

    // Changes default input border radius (unit: px)
    inputBorderRadius:  @Number - [Default: 0]

    // Changes default input background color
    inputBackgroundColor:  @String - [Default: '#fff']

    // Changes default input border size (unit: px)
    inputBorderSize:  @Number - [Default: 1]

    // Changes default input focus border color
    inputFocusBorderColor:  @String - [Default: '#dce4e8']

    // Changes default input focus shadow color
    inputFocusShadowColor:  @String - [Default: 'rgba(102, 175, 233, 0.6)']
